# Currency Icons

This directory contains currency icons for the game HUD.

## Required Assets

Please add the following PNG files to this directory:

1. **ninjapearl.png** - Ninja Pearl currency icon (premium currency)
2. **ryo.png** - Ryo currency icon (standard currency)
3. **shinobite.png** - Shinobite currency icon (gacha currency)

## Recommended Specifications

- Format: PNG with transparency
- Size: 32x32px or 64x64px (will be scaled to ~24px in HUD)
- Style: Should match the game's visual aesthetic
- Background: Transparent

## Usage

These icons are displayed in the top bar HUD overlay along with the current currency amounts.
